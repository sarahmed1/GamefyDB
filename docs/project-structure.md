# Project Structure (File and Folder Map)

This document describes every relevant folder and file currently present in the workspace.

## Root

- .git/ : Git metadata.
- .gitignore : Ignore rules for generated and local files.
- .pytest_cache/ : Pytest cache artifacts.
- .venv/ : Local Pyth   on virtual environment.
- AGENTS.md : Project agent guidance and stack recommendations.                             
- build/ : PyInstaller build artifacts.
- data/ : Input fixture data (Excel and/or legacy HTML).
- dist/ : Built distribution artifacts.
- GamefyDB.spec : PyInstaller build specification.
- local.db : Local SQLite database file.
- README.md : User-facing project overview and usage.
- requirements.txt : Python dependency constraints.
- src/ : Application source code.
- tests/ : Automated tests.

## src/

- main.py : Thin launcher that routes GUI vs CLI based on args.
- backend/ : Extraction, normalization, schema, and DB logic.
- frontend/ : PySide6 desktop UI.

## src/backend/

- __init__.py : Package marker.
- __pycache__/ : Python bytecode cache.
- app/ : Backend executable interfaces.
- database/ : SQLAlchemy ORM models and session helpers.
- pipeline/ : Core ETL pipeline modules.
- schemas/ : Pydantic validation schemas.

### src/backend/app/

- __init__.py : Package marker.
- __pycache__/ : Python bytecode cache.
- cli.py : CLI interface and run wrapper.

### src/backend/database/

- __init__.py : Package marker.
- __pycache__/ : Python bytecode cache.
- models.py : ORM table models for sessions, cash_reports, stock_reports, member_reports.
- session.py : Database initialization, session generator, and empty_database utility.

### src/backend/pipeline/

- __init__.py : Package marker.
- __pycache__/ : Python bytecode cache.
- extractor.py : File parsing and raw record extraction for Excel and legacy HTML.
- normalizer.py : Row normalization, type parsing, noise filtering, schema creation.
- orchestrator.py : Batch orchestration and transactional persistence.

### src/backend/schemas/

- __init__.py : Package marker.
- __pycache__/ : Python bytecode cache.
- record.py : Pydantic schemas aligned with ORM model fields.

## src/frontend/

- __init__.py : Package marker.
- __pycache__/ : Python bytecode cache.
- app/ : GUI app entrypoint.
- ui/ : Main window and background worker.

### src/frontend/app/

- __init__.py : Package marker.
- __pycache__/ : Python bytecode cache.
- gui.py : QApplication bootstrap and MainWindow launch.

### src/frontend/ui/

- __init__.py : Package marker.
- __pycache__/ : Python bytecode cache.
- main_window.py : Main desktop interface and database explorer tab.
- worker.py : QThread worker that runs pipeline asynchronously.

## tests/

- __pycache__/ : Python bytecode cache.
- test_empty_database.py : Verifies full-table cleanup and UTC extracted_at defaults.
- test_orchestrator_dedupe.py : Verifies rerun-safe behavior for same source file.
- test_pipeline_e2e.py : End-to-end extraction and normalization validation.

## data/

- CASH DATA 01-09-2025.html : Cash report fixture.
- MEMBER DATA 01-09-2026.html : Member report fixture.
- SESSION DATA 01-09-2025.html : Session report fixture.
- STOCK DATA 01-09-2025.html : Stock report fixture.

## build/

- GamefyDB/ : Intermediate PyInstaller analysis/build outputs.

### build/GamefyDB/

- Analysis-00.toc : PyInstaller analysis table of contents.
- base_library.zip : Python base library bundle generated for build.
- COLLECT-00.toc : PyInstaller collect stage manifest.
- EXE-00.toc : PyInstaller executable stage manifest.
- GamefyDB.exe : Built executable artifact (build stage).
- GamefyDB.pkg : Packaged payload used by executable.
- localpycs/ : Compiled Python cache used during bundling.
- PKG-00.toc : Package stage manifest.
- PYZ-00.pyz : Zipped Python bytecode archive.
- PYZ-00.toc : PYZ stage manifest.
- warn-GamefyDB.txt : Build-time warning output.
- xref-GamefyDB.html : Cross-reference report generated by PyInstaller.

## dist/

- GamefyDB/ : Distribution output folder.

### dist/GamefyDB/

- GamefyDB.exe : Distributed executable.
- local.db : Database file included in distribution output.
- _internal/ : Runtime support files generated by PyInstaller.
